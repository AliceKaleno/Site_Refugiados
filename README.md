# 🌍 Ponte Yarok

> Plataforma desenvolvida para conectar refugiados a serviços essenciais, promovendo inclusão, acolhimento e novas oportunidades.

---

## 📖 Sobre o Projeto

O Ponte Yarok foi desenvolvido como um projeto acadêmico durante as aulas de Desenvolvimento Web.

A proposta consistia em criar uma plataforma capaz de auxiliar refugiados em sua adaptação a uma nova comunidade, oferecendo acesso facilitado a serviços fundamentais como moradia, alimentação, educação e oportunidades de trabalho.

Além do desenvolvimento técnico, o projeto buscou abordar uma questão social relevante, utilizando a tecnologia como ferramenta de impacto positivo.

---

## ✨ Funcionalidades

### 🏠 Abrigos

* Informações sobre locais de acolhimento
* Apoio para refugiados em situação de vulnerabilidade

### 🍽 Alimentação

* Divulgação de centros de distribuição de alimentos
* Apoio alimentar para famílias refugiadas

### 💼 Empregos

* Divulgação de oportunidades de trabalho
* Incentivo à inserção no mercado profissional

### 📚 Educação

* Cursos e oficinas
* Capacitação para desenvolvimento pessoal e profissional

### 🎭 Eventos Culturais

* Incentivo à integração social
* Participação em atividades comunitárias

### ❤️ Doações

* Espaço para contribuição financeira
* Incentivo a ações solidárias
